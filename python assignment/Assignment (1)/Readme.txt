# Albus Security Internship Assignments

This repository contains my internship assignments completed during my tenure at Albus Security. The projects focus on practical applications of networking and Python, showcasing the skills and knowledge I have acquired.

## Assignments Overview

1. **Conditional Statements**: Programs demonstrating the use of if, if-else, and if-elif-else statements.
2. **Searching and Sorting Algorithms**: Implementations of Linear Search, Binary Search, Selection Sort, Merge Sort, and Quick Sort.
3. **Calculator Class**: A class capable of calculating the square and cube of a number.
4. **Programmer Class**: A class to store information about programmers working at Microsoft.
5. **Matrix Multiplication**: A function to perform matrix multiplication.

## Files

- `conditional_statements.py`: Contains examples of if, if-else, and if-elif-else statements.
- `searching_sorting.py`: Contains implementations of various searching and sorting algorithms.
- `calculator.py`: Contains the Calculator class with methods for square and cube calculations.
- `programmer.py`: Contains the Programmer class for storing programmer information.
- `matrix_multiplication.py`: Contains the matrix multiplication function.
